## What is Korex?

Korex is a free-to-use gaming platform where you can play, create, and share games without constantly hitting a wall that asks for your credit card. It's not trying to be Roblox with a coat of paint — it's a rethink of what a platform like this should actually look like in 2025.

We built it because we were frustrated with the same things you probably are. Stuff that used to be free getting locked behind Premium. Bans handed out for basically nothing with no real way to appeal. A chat filter that somehow blocks harmless words while actual bad stuff stays up for hours. We figured if we were going to complain about it, we might as well fix it.

## How It's Different

Here's a straight comparison — no marketing fluff:

| | Roblox® | Korex |
|---|---|---|
| Game costs | Some games charge real money | Games only cost Koru — no real money |
| Currency price | Robux | Koru costs half what Robux does |
| Privacy | Has had data leaks | Privacy is something we actually take seriously |
| Features | A lot is paywalled | Pretty much everything is free to use |
| Moderation | Bans for basically nothing | We don't throw bans around, and there's a real appeal process |
| Trying stuff | Pay upfront | Try models, plugins, games, etc. free for 3 days — timer pauses when you're not using them |
| Chat | Requires age verification | No age verification needed to chat |
| Themes | Need sketchy third-party extensions | Custom themes built right in |
| Code | Closed | Open source — you can see everything on GitHub |

## Features

- **Koru** — Our in-game currency. It costs half of what Robux does, and no game on the platform can charge real money.
- **KorexStudio** — A free desktop app for building your own games on PC. No subscription, no hidden costs.
- **Custom Themes** — Switch up how the platform looks without downloading anything sketchy.
- **Fair Moderation** — We don't ban first and ask questions never. There's an actual appeal process that works.
- **Free Trials That Make Sense** — Try any models, games, plugins, etc. free for 3 days. The timer only counts down when you're actively using it.
- **Open Source** — The code is right here. Read it, fork it, contribute to it, or just check that we're doing what we say we are.

We're still building. There's more coming — check the issues tab if you want to see what's in progress or suggest something.
